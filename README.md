# react-приложение для групп первого курса
Содержит в себе Калькулятор, Графику 2D и Графику 3D.

## Установка и запуск
```
npm install - установка зависимостей
npm start - запустить реакт на 3000 порту
```

## Основная ветка
```
react
```
